# Firmware

This directory will contain the ESP32 firmware for OpenHeading-IX.

## Planned modules

- HAL for sensors
- BMI323 driver
- BMM350 driver
- calibration engine
- sensor fusion engine
- Pixhawk MAVLink reference input
- NMEA2000 output
- diagnostics and logging

## Coding convention

Every source file shall contain a project header with:

- Project
- Version
- Author
- Date
- License
- Description
