# P0 I2C Scanner

First firmware milestone for M001 sensor bring-up.

## Purpose

Scan the ESP32 I2C bus on the fixed P0 pins:

- SDA: GPIO21
- SCL: GPIO22

## Expected result

The BMI323 and BMM350 Shuttle Boards appear at their I2C addresses after correct wiring and voltage verification.

## Status

Skeleton only. Firmware source will be added after the Shuttle Board pinouts are documented.
