# BG6 Lab Tool

Project: OpenHeading-IX  
Status: Concept

## 1. Function

A PC-side laboratory tool shall visualize and record OpenHeading-IX development data.

## 2. Planned views

- raw accelerometer values
- raw gyroscope values
- raw magnetometer values
- magnetometer circle plot
- pitch and roll
- heading
- calibration state
- Pixhawk comparison values
- OpenHeading-IX versus Pixhawk deviation

## 3. Candidate technologies

- Processing for live visualization
- Python for offline analysis
- CSV for raw data exchange

## 4. Open tasks

- Define serial protocol.
- Define CSV log format.
- Implement first raw data viewer.
