# Validation

Project: OpenHeading-IX

## Reference

The only external reference system for P0/P1 validation is the available Pixhawk Pro with GPS/compass.

## Validation targets

| Quantity | Target |
|---|---|
| Static heading | to be defined after first measurements |
| Dynamic heading | to be defined after first measurements |
| Pitch | to be defined after first measurements |
| Roll | to be defined after first measurements |
| Yaw rate | to be defined after first measurements |

## Method

OpenHeading-IX and the Pixhawk reference shall be mounted on the same rigid, non-magnetic carrier for comparison measurements.
