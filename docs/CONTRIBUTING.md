# Contributing

OpenHeading-IX is currently in early laboratory development.

## Contribution principles

- Document the reason for every relevant change.
- Keep code modular.
- Do not bypass documented measurement stages.
- Prefer reproducible measurements over assumptions.
- Keep hardware-specific code separate from algorithms.

## Commit style

Use clear commit messages, for example:

- docs: add BG1 sensor notes
- firmware: add P0 I2C scanner
- lab: record M001 results
- hardware: add first KiCad symbols
