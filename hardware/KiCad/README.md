# KiCad Hardware

This directory will contain KiCad projects for OpenHeading-IX.

## Planned hardware versions

- P0: Shuttle Board prototype, no custom PCB
- V1: first custom sensor and mainboard hardware
- V2: optimized compact hardware

## Rules

- KiCad 8 is the preferred CAD tool.
- Schematics and PCB layouts shall be documented in the docs directory.
- Design decisions shall be traceable to measurements or explicit engineering reasoning.
